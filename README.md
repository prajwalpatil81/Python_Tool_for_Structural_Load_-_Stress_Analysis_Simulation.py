# Structural Load & Stress Analysis Simulation (Python)

A Python-based structural engineering simulation tool for analyzing loads, stresses, and deflections in beam elements using classical Euler–Bernoulli beam theory.  
The project emphasizes engineering accuracy, modular design, clean code principles, and automated testing.

---

## Features
- Analysis of common beam configurations
- Bending stress and deflection calculations
- Support for point loads and uniformly distributed loads (UDL)
- Modular, object-oriented architecture
- Input validation and engineering safety checks
- Automated unit testing for calculation verification

---

## Engineering Scope

### Beam Types
- Simply Supported Beam
- Cantilever Beam

### Load Types
- Point Load
- Uniformly Distributed Load (UDL)

### Assumptions
- Linear elastic behavior
- Small deflections
- Static loading
- Euler–Bernoulli beam theory
